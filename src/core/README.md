# Core Module

Contains core application configuration, setup, and infrastructure.

## ?? Structure

- `api/` - HTTP client and API configuration
- `config/` - App configuration (i18n, React Query, etc.)
- `providers/` - React context providers
- `router/` - Routing configuration
- `store/` - Global Zustand stores
- `types/` - TypeScript type definitions

## ?? Purpose

Core module handles:
- API communication
- App initialization
- Global state management
- Routing
- Type safety
